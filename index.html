Page Notes & Design Decisions
Priority

Hotline number above the fold

The number 408-290-1144 appears in 3 places on the homepage: the nav CTA button, the hero, and the gold hotline banner. Someone on a phone in a stressful moment should never have to search for it.

Design

Gold = emergency, Navy = trust

Gold draws the eye to the most urgent actions (hotline, call CTAs). Navy provides authority and stability. The light gold (#FFFBB6) is used for background tints only — too light for text.

Content

3-path cards replace jargon

Instead of listing "services," the three cards ask the visitor what's happening to them right now. Plain language, no immigration jargon. Each card links to the Get Help page.

Bilingual

EN | ES toggle in navbar

Squarespace supports multiple language pages. The toggle in the top nav should be visible at all times — not hidden in a dropdown. Spanish version would mirror the same layout exactly.

When should I call?
🚨 ICE Activity Nearby

Page Notes
Priority

Red hero = urgency signal

This is the only page that uses red as the hero background. It signals urgency without being alarming — people landing here are already in a stressful situation and need to feel they're in the right place fast.

Bilingual — Highest Priority

Translate this page first

The Spanish version of Get Help should be the very first thing built. The /es/obtener-ayuda page should be identical in layout. Lets consider making the default language detection route Spanish-speaking users here automatically.

Mobile

Phone number as tappable link

On mobile, the hotline number must be an <a href="tel:4082901144"> link so it opens the dialer immediately. On Squarespace this is set in the button/link settings — do not use plain text for the number.

Content

KYR section is shareable

Lets consider adding a "Share this page" button below the KYR section, or a downloadable KYR card PDF. Community members often want to forward this to family members who may not speak English.

Page Notes
Priority

How to maintain the updates feed

The updates feed can be built two ways in Squarespace: (1) Use the Blog page type — each post is a blog entry with a category tag (Confirmed / Cleared / Resource). Easy to update, searchable, and automatically ordered by date. (2) Embed your Instagram feed directly. Option 1 is more flexible and bilingual-friendly. Option 2 requires zero extra effort since you're already posting to Instagram.

Content

What NOT to post

Never post specific home addresses, names of individuals detained, photos of community members, or real-time location data that could be used by ICE to identify targets. General area (neighborhood or city) is the right level of detail. This is consistent with your operational manual's community communication policy.

Bilingual

Spanish Updates page is essential

This page alongside Get Help should be the first two translated. Community members who most need this information are most likely to be Spanish-speaking. A simple /es/actualizaciones page with the same layout works well.

Nav

"Updates" in the main nav

This page earns a spot in the main navigation — it's a destination people will return to repeatedly, not just visit once. Placing it between Get Involved and the language toggle keeps the nav from feeling cluttered while making the page discoverable.

Gold (#F2C422)
Use for: primary CTA buttons, hotline banner background, section accents, stat numbers. Never use for body text — fails contrast on white backgrounds.
Navy (#0B2346)
Use for: navbar, footer, hero backgrounds, headings, body text. The primary trust color. Dominant on dark sections.
Light Gold (#FFFBB6)
Use for: background tints only (event tags, highlight boxes). Too light for text or CTAs. Never use as a button color.
Alert Red (#C0392B)
Use sparingly: Get Help page hero, urgency indicators only. Should feel like an alarm — loses impact if overused.
Off-white (#F8F6F0)
Use for: page section alternating backgrounds, card backgrounds. Warmer than pure white — feels community-centered rather than clinical.

📐 Squarespace Build Notes

Hotline CTA
Add the hotline number as a sticky button or announcement bar at the very top of every page. In Squarespace: Website → Pages → Website Tools → Announcement Bar.
Bilingual pages
Use Squarespace's built-in "Translated pages" feature or duplicate pages with /es/ URL prefix. Add the EN | ES toggle as a navigation link. Keep the same page structure — swap text only.
Events block
Use Squarespace's native Events page type for the trainings on Get Involved. Allows easy staff updates without touching code. Each event: title, date/time, location, description, registration link.
Donate button
Link to your external donation page. Set to open in a new tab (target="_blank"). In Squarespace: Button block → URL → check "Open in new window."
Phone number links
Any instance of the hotline number should be a Button or Link block with URL: tel:4082901144 — this makes it tappable on mobile. Do not put the number in plain text.
Social media
Instagram: instagram.com/scc_rapidresponsenetwork · Facebook: facebook.com/RRNSCC. Add via Squarespace's Social Links block in the footer on every page.
Logo placement
Official logo goes in the navbar (left side) and footer. In Squarespace: Design → Logo & Title. Upload as PNG with transparent background for best results on dark navy backgrounds.
Contact/Media page
Create as a standard page at /contact. Link from footer on every page. Optionally add a small "Media" text link in the nav. Never route media emails to the hotline or to outreach@ — keep communications@ exclusively for press.
Instagram embed
Squarespace supports Instagram feed embeds via the Instagram Block. Go to Edit Page → Add Block → Instagram. Connect your account (scc_rapidresponsenetwork). Shows latest 4–8 posts automatically — no manual updates needed.
Downloadable PDFs
Upload PDFs (hotline card, KYR card, legal referral list) via Squarespace Files. Then link each Download button to the file URL. PDFs appear as downloads on mobile and open in browser on desktop.
